# Khao Peeo - Restaurant Management System

A comprehensive SaaS solution for restaurant management, featuring:

- **Digital Menu & Ordering**: QR-based ordering for customers.
- **Admin Dashboard**: Real-time order tracking, billing, and analytics.
- **Kitchen Display System**: Streamlined order flow for kitchen staff.
- **Super Admin Panel**: Platform-level oversight for multi-tenant management.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Node.js, Express, MongoDB

## Getting Started

### Prerequisites

- Node.js (v16+)
- MongoDB (Local or Atlas)

### Installation

1.  **Backend Setup**
    ```bash
    cd backend
    npm install
    # Create .env file with your credentials
    npm start
    ```

2.  **Frontend Setup**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
